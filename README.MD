Hello !
Welcome to this template for a discord.js BOT ! You will find in this template an index.js file and an command Handler !
Before starting :
1) Install Node.js
2) Open CMD and type :
    - cd {yourdirectory}
    - npm i discord.js
3) Change the config.json
4) Start the bot
5) Enjoy !
